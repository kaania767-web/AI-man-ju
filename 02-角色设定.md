# 《浪漫主义流浪家》角色设定文档

---

## 角色一：郑新

### 基本信息
| 项目 | 内容 |
|------|------|
| 年龄 | 约 29-30 岁 |
| 身高 | 约 178cm |
| 体型 | 偏瘦，精干 |
| 气质 | 少年感与沧桑感并存，眼神里有故事 |

### 外貌特征
- **脸型**：轮廓分明，略微棱角，颧骨不高不低
- **眼睛**：有神的单眼皮或内双，笑起来有笑纹
- **发型**：略微凌乱的短发，自然黑色，带一点自然卷
- **皮肤**：日晒后的健康小麦色
- **胡茬**：若有若无的胡茬，不是刻意留的

### 服饰风格
- **现在的郑新**：简单舒适的棉麻衬衫/素色T恤 + 工装裤/旧牛仔裤 + 帆布鞋。整体色调偏大地色系（卡其、深灰、藏蓝）。身上没有多余的装饰，背包看起来用了很多年。
- **高中时的郑新**：校服或宽松T恤 + 运动裤，比现在更瘦小，眼神更锋利。

### 关键视觉标志
- 笑起来眼睛会眯成一条缝，有感染力
- 站姿放松，经常把手插在口袋里
- 说话时会微微歪头

### 角色关键词
流浪者 / 通透 / 少年气 / 风尘仆仆 / 眼里有光

## 角色二：「我」（鸡屎哥）

### 基本信息
| 项目 | 内容 |
|------|------|
| 年龄 | 约 29-30 岁 |
| 身高 | 约 175cm |
| 体型 | 普通身形，略微偏瘦 |
| 气质 | 温和内敛，观察者气质 |

### 外貌特征
- **脸型**：偏圆润，亲和力强
- **眼睛**：双眼皮，眼神温和
- **发型**：普通的短发，比郑新整齐一些
- **皮肤**：偏白（不常在外奔波）
- **眼镜**：有时戴眼镜

### 服饰风格
- **旅行中**：休闲衬衫 + 长裤 + 运动鞋，比郑新整洁
- 色调偏暖（米白、浅蓝、卡其）

### 关键视觉标志
- 习惯性沉默，表情在听多于在说
- 面对郑新时经常露出"拿你没办法"的微笑
- 动作节奏比郑新慢半拍

### 角色关键词
叙事者 / 细腻 / 旁观 / 温暖 / 活在常规里但向往自由
